eco-knock AI 서버

에코노베이션 동아리 챗봇 AI 서버 API 명세서입니다.
동아리 문서 기반 RAG 챗봇으로, 텍스트 및 이미지/PDF 입력을 지원합니다.

HTTPS
외부 연동 시 /chat 엔드포인트만 사용해주세요.
🔒 잠김 엔드포인트(/retrieve, /documents/index-all)는 AI 서버 내부 관리용으로, 외부에서 호출하지 않아도 됩니다.
GET /health

서버 정상 작동 여부를 확인합니다.

응답 예시
200 OK 
{
  "status": "ok"
}
POST /chat

챗봇에 질문합니다. 동아리 문서 기반으로 답변하며, 이미지 또는 PDF 파일을 함께 첨부할 수 있습니다.

Content-Type
multipart/form-data  # JSON이 아닌 form-data 형식
요청 파라미터
파라미터 타입 필수 설명
question string 필수 질문 내용
file file 선택 첨부 파일 (이미지 또는 PDF)
지원 파일 형식
jpg jpeg png webp gif pdf
응답 필드
필드 타입 설명
answer string 챗봇 답변 텍스트
sources string[] 답변에 참고한 문서 경로 목록
used_retrieval boolean 문서 검색 사용 여부 (true: 문서 기반, false: 일반 대화)
응답 예시 — 문서 기반 답변
200 OK 
{
  "answer": "AM은 Active Member의 줄임말로, 현 학기에 활동 중인 회원이에요 ✨",
  "sources": ["data/econovation_rules.md"],
  "used_retrieval": true
}
응답 예시 — 일반 대화
200 OK 
{
  "answer": "안녕하세요! 에코노베이션 관련 질문이 있으면 뭐든 물어보세요 😊",
  "sources": [],
  "used_retrieval": false
}
에러 응답
상태 코드설명
400 지원하지 않는 파일 형식
500 서버 내부 오류
POST /retrieve 🔒 내부 관리용

벡터 DB 검색 결과만 반환합니다. AI 서버 디버깅 목적으로만 사용합니다.

Content-Type
application/json
요청 필드
필드타입필수기본값설명
question string 필수 – 검색할 질문
top_k integer 선택 3 반환할 검색 결과 수
응답 예시
200 OK 
{
  "question": "AM 활동 조건",
  "results": [
    {
      "chunk_id": 7,
      "title": "제2장 회원 > 제5조 회원의 분류 > 1. AM (Active Member)",
      "score": 0.884554,
      "text": "- 현 학기에 활동하는 ...",
      "source": "data/econovation_rules.md"
    }
  ]
}
POST /documents/index-all 🔒 내부 관리용

문서 파일들을 청킹 및 임베딩하여 벡터 DB에 인덱싱합니다.

요청 예시
Request Body 
{
  "file_paths": [
    "data/econovation_rules.md",
    "data/club_intro.md",
    "data/blog_dev.md"
  ]
}
응답 예시
200 OK 
{
  "total_files": 3,
  "total_chunks": 683,
  "results": [
    { "file_path": "data/econovation_rules.md", "chunks": 89, "error": null },
    { "file_path": "data/club_intro.md", "chunks": 12, "error": null },
    { "file_path": "data/blog_dev.md", "chunks": 288, "error": null }
  ]
}
